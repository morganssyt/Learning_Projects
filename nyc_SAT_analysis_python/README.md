# NYC SAT Scores Analysis

Analysis of SAT performance across 375 NYC public high schools.

## Dataset
- `schools.csv` - Contains school names, boroughs, and average SAT scores (math, reading, writing)

## Analysis
1. **Best Math Schools** - Schools with average math score >= 640
2. **Top 10 Schools** - Highest total SAT scores (math + reading + writing)
3. **Borough Comparison** - Identifies borough with highest SAT score standard deviation

## Requirements
- Python 3
- pandas
