# Video Games Sales Analysis (SQL)

SQL analysis exploring the golden age of video games using sales data and critic/user ratings.

## Dataset
Source: [DataCamp - When Was the Golden Age of Video Games?](https://projects.datacamp.com/projects/2013)

**Tables:**
- `game_sales` - Video game sales data
- `critics_avg_year_rating` - Average critic scores by year
- `users_avg_year_rating` - Average user scores by year

## Queries
- Best selling games of all time
- Top 10 years according to critics (with 4+ games released)
- Golden years where critics or users rated games above 9
